@rqm Support for the hall effect sensor was removed from Espressif's IDF in version 5:

https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/migration-guides/release-5.x/5.0/peripherals.html#api-changes

https://github.com/micropython/micropython/commit/2af229c3cc55ceefc4cfba0f50092af725c3bfa9

Hope that helps! BTW the built-in hall effect sensor didn't perform very well anyway... 😜
